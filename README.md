# ParticleCannon
Particle Cannon COSC 3P98 A3
Cameron Cronheimer (6517080), Aman Braich (6511679)

----------MODES----------
Q: Quit
R: Reset
F: Manual Firing
O: Single Shot
K: Spin
M: Friction
L: Lighting
C: Culling
B: Bouncy

to build ->  g++ -framework OpenGL -framework GLUT -o cannon cannon.cpp
to run -> ./cannon
